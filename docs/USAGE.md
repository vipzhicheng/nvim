# 帮助文档

本文档建立在当前配置的基础上，是结合安装的插件和快捷键绑定之后的使用手册。快捷键绑定会经常变化，原因大概是之前不了解的内置快捷键后来发现有用，就给让出来，或者发现更顺手的键位。

## 为插件定义的快捷键

### 基于 `quickui`

| key | usage    |
|-----|----------|
| K   | 右键菜单 |
| M   | 顶部菜单 |

### 基于 `coc`

| key | usage                |
|-----|----------------------|
| C-p | 当前目录查找文件     |
| C-b | 在缓冲区查找文件     |
| C-e | 在最近访问文件中查找 |
| C-f | 查找当前文件中的函数 |

### 基于 `rnvimr`

| key | usage                   |
|-----|-------------------------|
| R   | 在 `vim` 中打开`Ranger` |

### 基于 `MarkdownPreview`

| key        | usage         |
|------------|---------------|
| `<leader>`pv | 预览 Markdown |

### 基于 `Airline`

| key | usage              |
|-----|--------------------|
| rr  | 随机更换状态栏主题 |

无聊的时候逗手指用的

### 基于 `Undotree`

| key | usage      |
|-----|------------|
| U   | 打开撤销树 |

### 基于 `goyo`

| key | usage        |
|-----|--------------|
| gy  | 切换专注模式 |

### 基于 `easy-align`

| key | usage            |
|-----|------------------|
| ga  | 根据指定符号对齐 |

**注：这里后面需要跟对应的符号，比如常用的`=`**
